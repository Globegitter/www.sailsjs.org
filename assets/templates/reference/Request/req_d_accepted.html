req.accepted

Purpose

Return an array of Accepted media types ordered from highest quality to lowest.

Example Usage

console.log(req.accepted);

/*
  [ { value: 'application/json',
      quality: 1,
      type: 'application',
      subtype: 'json' },
  { value: 'text/html',
       quality: 0.5,
       type: 'text',
       subtype: 'html' } ]
*/
